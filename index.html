BK Minecraft Server

Used React and Bootstrap to create a server information page for my friend's Minecraft server. It uses an API that grabs server information from Minecraft. It then updates to display the message of the day, some basic information for connecting, online status, players online count, and players online list. Below that is a DynMap embed for tracking players live on the server map.
